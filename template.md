# B2B Category Wiki - LLM Extraction Template

## INSTRUCTIONS FOR LLM

You are tasked with extracting information from provided documents (seller catalogs, PDFs, transcripts) to populate this category wiki. Follow these strict rules:

**CRITICAL RULES:**
1. **ONLY extract information explicitly stated in the provided documents**
2. **DO NOT use your general knowledge or training data**
3. **DO NOT make assumptions or inferences beyond what is directly stated**
4. **If information is not found in the documents, mark the field as "NOT FOUND IN SOURCE"**
5. **Always cite the source document and page/section where information was found**
6. **Use exact terminology from the source documents**
7. **For numerical data, quotes, or specifications - extract verbatim**
8. **If documents conflict, note both versions with sources**

**Output Format:**
- Use this exact structure
- Fill each section based only on source documents
- Include [SOURCE: document_name, page/section] after each extracted piece of information
- Use "NOT FOUND IN SOURCE" for missing information
- Use "CONFLICTING INFORMATION" when sources disagree, then list all versions
